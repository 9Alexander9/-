Инструкция пользователя: Expense Tracker
1. Запуск приложения
Откройте терминал (командную строку) в папке с проектом и введите:

python main.py

2. Добавление расхода
Заполните поля в верхней части окна:
Сумма: Введите число (например, 500). Только положительные значения.
Категория: Выберите тип расхода из списка (Еда, Транспорт и др.).
Дата: Укажите дату в формате ДД.ММ.ГГГГ (по умолчанию стоит сегодня).
Нажмите кнопку «Добавить».
⚠️ Важно: Если формат даты или суммы неверен, программа покажет ошибку.
3. Просмотр и фильтрация
Все записи отображаются в таблице.
Для фильтрации используйте список «Фильтр по категории» над таблицей.
Поле «Итого» справа показывает сумму расходов. Если включен фильтр, считается сумма только по выбранной категории.
4. Удаление записи
Кликните по нужной строке в таблице.
Нажмите кнопку «Удалить выбранное».
5. Сохранение данных
Все данные автоматически сохраняются в файл expenses.json в папке с программой. При следующем запуске история загрузится автоматически.
